# Guna Job AI 🌍

## Project Goal

Guna Job AI is a personal AI assistant that helps Guna find South Indian Chef jobs worldwide.

### Features

- 🌍 Search jobs globally
- 🔔 Instant mobile notifications
- 📄 Resume matching
- 🤖 AI-generated application answers
- ✅ Visa sponsorship filter
- 📊 Job application tracker
